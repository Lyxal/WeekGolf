## General informations about WeekGolf
- WeekGolf is a website about <a href="https://en.wikipedia.org/wiki/Code_golf">code golf</a>.<br>
WeekGolf is not the only website about code golf.


- What makes WeekGolf special is that every week there is a new challenge.<br>
At the end of the week, the answers are made public, so that everyone can progress.<br>

- The community votes each month to add a new programming language to WeekGolf

Information on how WeekGolf is made:
- Front end: <a href="https://www.solidjs.com/">Solid.JS</a>, <a href="https://sass-lang.com/">Sass</a>, <a href="https://www.typescriptlang.org/">TypeScript</a>, <a href="https://vitejs.dev/">Vite</a>
- Back end: <a href="https://www.typescriptlang.org/">TypeScript</a>, <a href="https://nodejs.org/en">Node.JS</a>, <a href="https://expressjs.com/">Express</a>
- DataBase: <a href="https://mariadb.org/">MariaDB</a>

The person who created WeekGolf and maintained it alone for months is <a href="https://github.com/aderepas">TKirishima</a>.


## Contributing to WeekGolf
<strong>Anyone</strong> can contribute to improve WeekGolf!<br>
However, when I talk about improving WeekGolf, it's not about making the code itself as small as possible :)!<br>
If you know how to improve the code of WeekGolf from a performance, readability or code quality point of view, don't hesitate to talk about it on <a href="https://discord.gg/CqrbqSkdX3">discord</a> or to make a PR directly on this repo
